# 📖 WordPop

**基于艾宾浩斯遗忘曲线的 Windows 桌面弹窗背单词工具**

[![Latest Release](https://img.shields.io/github/v/release/linianchen666/wordpop?label=latest)](https://github.com/linianchen666/wordpop/releases/latest)
[![Platform](https://img.shields.io/badge/platform-Windows-blue)](https://github.com/linianchen666/wordpop/releases/latest)

---

## ✨ 特性

- **艾宾浩斯遗忘曲线** — 9 阶段间隔复习（1min → 5min → 30min → 4h → 1d → 2d → 4d → 7d → 15d → 已掌握）
- **两阶段交互** — 先看单词主动回忆，再点击显示释义，训练真正的记忆能力
- **三种反馈精确匹配** — 不认识（回退阶段1）/ 模糊（阶段不变）/ 认识（阶段+1）
- **连续熟知机制** — 第1次熟知→15天后复习，第2次连续熟知→彻底已掌握
- **非侵入式弹窗** — 小窗口始终置顶，不影响正常工作
- **连续刷词** — 点完一个单词立即弹出下一个，无需等待
- **完整词库内置** — CET-4（4544词）、CET-6（3991词）、考研词汇（5047词）
- **随机排序** — 单词出现顺序完全随机
- **每日词量自定义** — 支持 5/10/20/30/50 个新词/天
- **系统托盘常驻** — 最小化到托盘，随时暂停/恢复
- **键盘快捷键** — 全键盘操作，无需鼠标
- **点击发音** — 基于 Web Speech API
- **学习统计** — 今日/累计数据、连续打卡、阶段分布
- **安装进程检测** — 安装新版本时自动检测并关闭旧进程
- **数据安全** — 升级版本学习进度完全保留

## 🚀 下载安装

1. 前往 [Releases](https://github.com/linianchen666/wordpop/releases/latest) 下载最新版 `WordPop Setup X.X.X.exe`
2. 双击运行安装程序
3. 如果旧版 WordPop 正在运行，安装程序会自动提示关闭
4. 安装完成后启动，首次运行会弹出设置向导
5. 选择词库和每日词量，开始背单词

> 学习数据存储在 `%AppData%\wordpop\`，卸载重装不会丢失

## 🎮 使用方法

| 快捷键 | 回忆阶段 | 显示阶段 |
|--------|---------|---------|
| 空格 | 显示释义 | 发音 |
| → / D | 显示释义 | 认识 |
| ↓ / S | — | 模糊 |
| ← / A | — | 不认识 |
| Enter | 显示释义 | 认识 |
| M | — | 熟知 |
| Esc | 最小化 | 最小化 |

### 鼠标操作

| 操作 | 说明 |
|------|------|
| 点击单词 | 朗读发音 |
| 点击「显示释义」 | 展开翻译和操作按钮 |
| 点击 ★ (右上角) | 熟知：标记为已掌握 |
| 右键托盘图标 | 暂停/恢复/设置/统计 |

## 🧠 复习机制

### 艾宾浩斯 9 阶段间隔

| 阶段 | 间隔 | 说明 |
|------|------|------|
| 1 | 1 分钟 | 快速确认 |
| 2 | 5 分钟 | 短期记忆 |
| 3 | 30 分钟 | 同次使用期间 |
| 4 | 4 小时 | 当天晚些时候 |
| 5 | 1 天 | 跨天复习 |
| 6 | 2 天 | |
| 7 | 4 天 | |
| 8 | 7 天 | |
| 9 | 15 天 | 长期巩固 |
| — | 不再推送 | 已掌握 |

### 三种操作行为

| 操作 | 效果 | 适用场景 |
|------|------|---------|
| **✗ 不认识** | 回退到**阶段 1** | 完全没印象，从头开始 |
| **~ 模糊** | 阶段**不变**，重新进复习队列 | 有印象但不确定 |
| **✓ 认识** | 阶段 **+1** | 确实认识，正常推进 |

### 熟知机制

- **第 1 次点熟知** → 跳到阶段 8（15 天后再出现）
- **第 2 次连续熟知** → 直接标记为**已掌握**，不再出现
- 中间如果点了认识/模糊/不认识，熟知计数清零，需重新累计 2 次

## 🛠 技术栈

- **Electron 28** — 桌面应用框架
- **better-sqlite3** — 本地 SQLite 数据库（WAL 模式）
- **原生 HTML/CSS/JS** — 轻量渲染，无框架依赖
- **electron-builder** — 打包与分发（NSIS 安装程序）
